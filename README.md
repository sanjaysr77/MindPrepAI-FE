# MindPrep AI - Frontend

A comprehensive placement preparation platform built with React, TypeScript, and Tailwind CSS. Features AI-powered interview practice, resume analysis, and personalized learning paths.

## Features

- **Quiz System** - Subject-based quizzes with progress tracking
- **AI Interview Practice** - Role-based mock interviews with voice recording and real-time feedback
- **Resume Analyzer** - AI-powered resume evaluation and suggestions
- **Personalized Reports** - Detailed performance analytics and insights
- **Company-Specific AI** - Tailored interview prep for target companies
- **Subject-Specific AI** - Deep-dive learning for specific topics
- **Dashboard** - Centralized hub for all learning activities

## Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **Vite** - Build tool
- **React Router 7** - Navigation
- **Axios** - HTTP client
- **React Query** - Data fetching
- **Three.js & React Three Fiber** - 3D visualizations
- **Framer Motion** - Animations
- **Recharts** - Data visualization
- **React Hot Toast** - Notifications

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

## Project Structure

```
src/
├── components/          # Reusable UI components
├── pages/              # Page components (Dashboard, Quiz, etc.)
├── config/             # Configuration files
├── types/              # TypeScript type definitions
└── assets/             # Static assets
```
